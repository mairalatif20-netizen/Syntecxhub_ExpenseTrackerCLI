Expense Tracker CLI
Description

Expense Tracker CLI is a command-line based application developed in Python to help users manage their personal finances. The system allows users to record income and expense transactions along with details such as date, category, and amount. All records are stored permanently using a SQLite database or CSV files.

The application provides features to view transactions, generate monthly financial summaries, and export records into CSV or Excel format for reporting and analysis. It can also generate simple expense charts as image files for visual representation of spending patterns.

This project demonstrates important programming concepts including file handling, database management, date processing, data aggregation, and basic data visualization.

Features
Add income and expense entries
Store transaction records permanently
View all financial transactions
Generate monthly income and expense summaries
Export data to CSV and Excel files
Create expense charts for analysis
Technologies Used
Python
SQLite Database
CSV File Handling
Pandas
Matplotlib

The main purpose of this project is to provide a simple financial tracking system while demonstrating practical implementation of:

File I/O operations
Database connectivity

Output Files

The system can generate:
